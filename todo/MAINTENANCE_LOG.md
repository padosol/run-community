## 📝 새 유지보수 작업 추가 방법

1. 해당 날짜의 파일을 확인합니다 (예: `MAINTENANCE_LOG_2025-12-29.md`)
2. 파일이 없으면 새로 생성합니다
3. 템플릿을 사용하여 작업 내역을 추가합니다
4. 이 인덱스 파일(`MAINTENANCE_LOG.md`)에 날짜별 링크를 추가합니다
